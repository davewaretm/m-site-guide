# m-site-guide

## 项目简介

m-site-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面集合的存放与展示空间。

## 目录说明

```
m-site-guide/
├── index.html          # 入口页面
├── docs/               # 文档及资源文件
├── pages/              # 独立 HTML 页面存放目录
└── README.md           # 本文件
```

- **index.html**：仓库的默认入口文件，可用于展示页面索引或导航。
- **docs/**：存放与项目相关的文档、说明、图片等资源。
- **pages/**：各独立 HTML 页面均放置于此目录下，方便统一管理。

## 页面归档说明

- 本仓库中的所有 HTML 页面均为独立归档，彼此之间无依赖关系。
- 页面内容可涵盖各类信息展示、工具、参考等，但不会包含任何具体域名、网站链接或营销推广信息。
- 仓库维护者会根据实际需要，不定期新增或更新页面，以保持内容的时效性与准确性。

## 维护说明

- 本仓库由维护者个人管理，欢迎通过 Issues 或 Pull Requests 提交建设性意见或错误反馈。
- 所有页面均遵循静态 HTML 规范，不依赖后端服务或动态脚本。
- 如需引用或分发本仓库中的任何页面，请保留原始出处说明，并遵守相关开源许可协议。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件，即可在浏览器中查看对应页面。
3. 也可将本仓库部署至 GitHub Pages 或其他静态托管服务，实现线上访问。
